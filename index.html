Intersection Observers

Funktion:

Jedem Bild außerhalb des Sichtbaren Bereiches muss die Classe "obserbers-item" hinzugefügt werden.

< img src="..." class="observers-item" alt="..." >

Bei erreichen der Classe "observers-item" wird die Classe "show" hinzugefügt. Jene hat die CSS Eigenschaft langsam in den Sichtbaren Bereich zu fahren.

img.observers-item {
opacity: 0;
transition: all .5s ease;
visibility: hidden;
transform: translateY(15%);
}

img.observers-item.show {
opacity: 1;
visibility: visible;
transform: translateY(0);
}
